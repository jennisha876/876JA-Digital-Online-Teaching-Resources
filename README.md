876JA Digital Online Teaching Resources is a relatively new company that specializes in providing online teaching resources such as notes, tutorials etc. that subscribers can pay a fee to access and download.
We create a functional dynamic website using PHP for the company with the following guidelines.

MAIN GUIDELINES:
The website should include the following:
1. Homepage, customer service, about us, contact us, FAQ etc.
2. Online Registration Form – this will allow new users to register for an online account. The information from this form should be stored in a database.
3. Login Form – this form will be used to log in registered users based on credentials stored in the database.
4. Resource Catalog / Library – this page should allow visitors and subscribers to browse the available teaching resources. Resources should be filterable and searchable by category (e.g., Mathematics, Science, etc,). Six (6) categories are sufficient for this project.
5. Pricing / Subscription Plans – this page will clearly display the available subscription tiers (e.g., Basic, Pro, School License). For example, clicking the subscribe button should take the user to an online payment page.
6. Online Payments – the website must securely integrate with a payment gateway (e.g., Stripe, PayPal) to process subscription fees.
7. Links to at least three other relevant websites (social media sites, business partners etc.)
